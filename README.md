# Users API

## Description

This is a RESTful api that returns users and allow to create them

## Endpoints

- /users
    - POST: Creates a new user
    - GET: Lists all the current users

## Dependencies

- express

### Usage examples

- GET /users

``` json
{
   "name": "username" 
}
```
